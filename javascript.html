JavaScript

What is JavaScript?

JavaScript or (JS) is a high level programming language that runs along side html and css.
It is what turns a static web page or site into a dynamic one.

A Simple Example


                            let a = 10; 
                            let b = 20; 
  
                            console.log(a - b);

This is a simple script to subtract b from a.

Best Practices

  • Always declare local variables
  • Declare arrays with const
  • Use descriptive variable and function names
